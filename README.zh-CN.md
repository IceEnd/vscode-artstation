## 命令

| Name                      | Description                                 |
| :------------------------ | :------------------------------------------ |
| `Artstation login`        | 输入 Artstation 的 cookie 进行登录           |
| `Artstation start`        | 启动 Artstation                             |

## Config

| Name                            |      Type       |   Default    | Description                |
| :------------------------------ | :-------------: | :----------: | :------------------------- |
| `artstation.wallpaperPath`      |    `string`     |     `''`     | 壁纸保存路径                |
